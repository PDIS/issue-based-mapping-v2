# PDIS議題卡/議題分析表/議題手冊

## Requirement

* Node.js >= v8.x

## Install

```
git clone https://github.com/PDIS/issue-based-mapping-v2.git
cd issue-based-mapping-v2
npm install
```

## Start & watch

```
npm start
```

## Simple build for production

```
npm run build
```

## Update sources

```
git pull
npm install
```
